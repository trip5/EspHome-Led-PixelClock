EspHome-Led-PixelClock

ESP Web Tool

Flash Firmware

With this tool, you can:

  • Install & update firmware
  • Connect device to the Wi-Fi network
  • Visit the device's hosted web interface
  • Access logs and send terminal commands
  • Add the device to Home Assistant

Flashing Mode

Flashing is dead-simple. Hold the 'Download' button while powering-on the clock, or press the reset button while holding 'Download' button. Probably. Your clock may be different.

Improv Mode

To send Wi-fi configuration, visit the device's WebUI, or add to Home Assistant, reset without holding the 'Download' button. Then click 'Connect' again.

These firmwares likely contain Time Zones that do not match yours. That's OK, you can change them! For non-HA versions, you can change the POSIX string of the Alternate Time Zone in the device's Web UI. For HA versions, you can override the time zones with Helpers.
